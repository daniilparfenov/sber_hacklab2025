# High-Performance benchmark for Cryptography, Random Number Generation and CRC

> **Бенчмарк высокопроизводительных функций для генерации случайных чисел, шифрования AES256_ECB и вычисления CRC32**

Библиотека предоставляет набор высокооптимизированных функций для:

- **Генерации случайных чисел** с различными распределениями
- **Криптографического шифрования** AES256_ECB
- **Вычисления контрольных сумм** CRC32



## Сборка и тестирование

### Сборка проекта

```bash
# Скомпилировать все компоненты
make

# Очистить сборочные файлы
make clean
```

### Запуск тестов

```bash
# Запустить тесты корректности
./app correctness

# Запустить тесты производительности
./app performance

# Запустить все тесты
./app all
```
